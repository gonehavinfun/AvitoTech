1. Предусловия:
- Неавторизованный пользователь
- Desktop-версия сайта
- Chrome-браузер последней стабильной версии

2. Подготовка:
- Очистить кэш и куки браузера перед началом каждого теста

3. Тестирование добавления объявления в избранное:

Сценарий 1: Добавление избранного объявления со страницы объявления:
- Открыть браузер и перейти на главную страницу Avito
- Выполнить поиск объявления из разных категорий
- Открыть одно из найденных объявлений
- Нажать на кнопку "Добавить в избранное" на странице объявления
- Перейти на страницу "Избранное" (https://www.avito.ru/favorites)
- Проверить наличие добавленного объявления в списке избранных

ОР: Объявление успешно добавлено в список избранных на странице объявления. На странице "Избранное" присутствует добавленное объявление

Сценарий 2: Добавление избранного объявления со страницы поиска:
- Открыть браузер и перейти на главную страницу Avito
- Выполнить поиск объявлений из разных категорий
- Нажать на кнопку "Добавить в избранное" на карточке объявления в результате поиска
- Перейти на страницу "Избранное" (https://www.avito.ru/favorites)
- Проверить наличие добавленного объявления в списке избранных

ОР: Объявление успешно добавлено в список избранных на странице поиска. На странице "Избранное" присутствует добавленное объявление

4. Тестирование удаления объявления из избранного:

Сценарий 1: Удаление объявления из избранного со страницы "Избранное":
- Открыть браузер и перейти на страницу "Избранное" (https://www.avito.ru/favorites)
- Найти объявление в списке избранных
- Нажать на кнопку удаления объявления из избранного
- Проверить, что объявление исчезло из списка избранных

ОР: Объявление успешно удалено из списка избранных на странице "Избранное". Объявление больше не отображается в списке избранных

Сценарий 2: Удаление объявления из избранного со страницы объявления:
- Открыть браузер и перейти на главную страницу Avito
- Выполнить поиск объявления из разных категорий
- Открыть одно из найденных объявлений
- Проверить, что объявление уже добавлено в избранное
- Нажать на кнопку удаления объявления из избранного на странице объявления
- Перейти на страницу "Избранное" (https://www.avito.ru/favorites)
- Проверить, что объявление исчезло из списка избранных

ОР: Объявление успешно удалено из списка избранных на странице объявления. Объявление больше не отображается в списке избранных на странице "Избранное"

5. Тестирование отображения и работоспособности списка избранных:
- Открыть браузер и перейти на страницу "Избранное" (https://www.avito.ru/favorites)
- Проверить, что все добавленные объявления отображаются в списке избранных
Для каждого объявления проверить:
- Наличие основной информации (название, цена, категория и т.д.)
- Корректность отображения изображения (если есть)
- Доступность ссылки на страницу объявления для перехода
- Возможность удаления объявления из избранного

ОР: Все добавленные объявления отображаются в списке избранных.
Для каждого объявления:
- Основная информация (название, цена, категория и т.д.) присутствует и корректно отображается
- Изображение, если есть, отображается правильно
- Ссылка на страницу объявления доступна и корректно переходит на страницу объявления
- Возможность удаления объявления из избранного доступна и работает корректно

6. Переход на страницу объявления из избранных:
- На странице избранных объявлений найти объявление
- Кликнуть на объявление для перехода на страницу объявления
- Проверить, что открылась корректная страница объявления

ОР: Переход на страницу объявления из списка избранных осуществляется корректно, и открывается правильная страница объявления

7. Проверка разных категорий:
- Повторить шаги 2-6 для объявлений из оставшихся категорий, чтобы убедиться, что функциональность работает для всех категорий

ОР: Функциональность "Избранное" работает корректно для объявлений из разных категорий, и все сценарии добавления, удаления и проверки работают одинаково надежно

8. Тестирование на разных разрешениях экрана:
- Изменить разрешение экрана браузера на несколько значимых значений
- Пройти через основные шаги добавления, удаления и проверки объявлений в избранном на каждом разрешении

ОР: Функциональность "Избранное" успешно работает на разных разрешениях экрана, и элементы интерфейса отображаются корректно

9. Тестирование с плохим интернет-соединением:
- Ограничить скорость интернета с помощью инструментов разработчика браузера
- Проверить, как функциональность "Избранное" работает при плохом интернет-соединении

ОР: При плохом интернет-соединении функциональность "Избранное" сохраняет стабильность, и система адекватно обрабатывает действия пользователя

10. Тестирование на устойчивость:
- Повторить последовательность добавления и удаления объявлений несколько раз подряд
- Проверить, что функциональность остается стабильной и не возникает ошибок

ОР: Последовательное добавление и удаление объявлений не приводит к сбоям функциональности, и система остается стабильной

11. Финальное тестирование:
- Повторно пройти по всем шагам тестирования для проверки, что ранее обнаруженные дефекты были исправлены

ОР: Все шаги функционального тестирования успешно пройдены, ранее обнаруженные дефекты исправлены. Функциональность "Избранное" готова к выпуску
